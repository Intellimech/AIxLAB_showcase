## A chi è rivolto
Questo tutorial è pensato per ingegneri meccanici ed elettronici che vogliono iniziare a usare Python per sostituire Excel nelle analisi complesse.

## Programma
1.  **Setup Ambiente:** Installazione di Anaconda e VS Code.
2.  **Basi del linguaggio:** Variabili, Liste, Cicli For.
3.  **Libreria Pandas:** Come leggere file CSV ed Excel.
4.  **Matplotlib:** Creare grafici di tendenza e scatter plot.
